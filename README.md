# UC3-Aula-0---Arquivos-e-materiais-de-apoio

## Como usar os livros

Este repositório contém materiais de apoio em formato PDF para o estudo de JavaScript. Para aproveitar ao máximo os recursos disponíveis:

1. **Faça o download** dos arquivos PDF que deseja estudar clicando no nome do arquivo e depois em "Download"
2. **Leia os capítulos** conforme as atividades propostas
3. **Pratique os exemplos** apresentados nos livros para consolidar o aprendizado
4. **Use como referência** durante as aulas e exercícios práticos
